# Echo Engineering PLC — Website

> **Connecting Ethiopia · Powering Progress**
> Telecom Infrastructure & Electro-Mechanical Engineering Company

This repository contains the public-facing website for **Echo Engineering PLC** (legal name: Echo Electromechanical Engineering PLC), an Ethiopian engineering company specializing in telecom infrastructure deployment and electro-mechanical works.

## About the Company

- **Founded:** 07 April 2024
- **Headquarters:** Lideta Sub-City, Addis Ababa, Ethiopia
- **Services:** Telecom Infrastructure · Electro-Mechanical Works
- **Vendor expertise:** Nokia · Huawei · Ericsson · ZTE
- **Operator engagements:** Safaricom Ethiopia · Ethio Telecom
- **Certified training:** Huawei – Addis Ababa University ICT Practice Center

## Live Site

Deployed on Vercel — see the live URL in this repository's Deployments tab.

## Tech Stack

Pure static site — no build step required.

- HTML5 (single-file `index.html`)
- CSS3 (embedded)
- Vanilla JavaScript (embedded, no dependencies)
- SVG logo and brand assets

## Local Development

Just open `index.html` in any browser. No build, no install.

## Deployment

Auto-deploys to Vercel on every push to `main`.

Configuration in [`vercel.json`](vercel.json) handles:
- Clean URLs
- Long-term caching of SVG assets
- Standard security headers (X-Content-Type-Options, X-Frame-Options, Referrer-Policy)
